# gemini-newsletter-generator

A Flask application that's pulling from a dagster pipeline on the backend as a data source for the Reddit API to find relevant comments based on the user's query, then feeding the comments to Gemini via the gcloud vertex sdk in order to create a newsletter for the user to read

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/gemini-newsletter-generator.git
cd gemini-newsletter-generator
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
